Que es JS?.

es un lenguaje de programación ligero, interpretado, o compilado justo-a-tiempo (just-in-time) con funciones de primera clase.

Diseñado por: Netscape Communications, Fundación Mozilla

Para que sirve JS?..

Para hacer paginas web interactivas y desde :

  • actualizar fuentes de redes sociales.
  • mostrar animaciones.
  • mapas interactivos.
  • Etc..

Ejemplos de JS..

Por ejemplo, el siguiente código define una función simple llamada square ("cuadrado"):

function square(number) { return number * number; }
